# LUMINA.STUDIO

Вебсайт фотостудії LUMINA.STUDIO.

## Реалізовано

- головну сторінку сайту;
- сторінку послуг;
- сторінку залів;
- сторінку фотографів;
- сторінку контактів;
- реєстрацію та авторизацію користувачів;
- кабінет клієнта;
- кабінет фотографа;
- адмін-панель;
- створення та керування бронюваннями;
- підтвердження, оплату, скасування та видалення бронювань;
- графік роботи фотографів;
- заявки на зворотний дзвінок;
- показники роботи фотостудії.

## Технології

- Node.js
- Express.js
- MySQL
- HTML
- CSS
- JavaScript

## Запуск проєкту

1. Встановити залежності:

npm install

2. Імпортувати базу даних з файлу:

database/lumina.sql

3. Запустити сервер:

node server.js

4. Відкрити сайт у браузері:

http://localhost:3000

## Структура проєкту

public/      - HTML, CSS, JS та зображення сайту
database/    - дамп бази даних
docs/        - ER-діаграма та файл курсової роботи
scripts/     - допоміжні скрипти
server.js    - серверна частина
